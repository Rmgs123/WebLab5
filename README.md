# WebLab5

## Technological Stack

- **SPA** - vue3 + typescript + design system
- **REST** - DRF
- **Docker**
- **CI** - github actions/buildbot/etc...
- **Websockets**

## Backend start

```bash
cd backend
python manage.py migrate
python manage.py runserver
```

## Frontend start

```bash
cd frontend
npm install
npm run serve
```
